# ENGULFING-CANDLE-STRATEGY
Logic used:

Bullish Engulfing → Buy

Previous candle is bearish

Current candle is bullish

Current candle body fully engulfs the previous candle body

Bearish Engulfing → Sell

Previous candle is bullish

Current candle is bearish

Current candle body fully engulfs the previous candle body

The EA opens one trade per signal and uses Stop Loss and Take Profit inputs.
